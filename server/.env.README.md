# 环境配置示例文件

本项目使用三个环境：Local（本地）、Dev（开发）、Prod（生产）

## 配置文件说明

### 前端配置

- `.env.local` - 本地开发环境
- `.env.development` - 开发服务器环境
- `.env.production` - 生产服务器环境

### 后端配置

- `.env.local` - 本地开发环境
- `.env.dev` - 开发服务器环境
- `.env.production` - 生产服务器环境

## 当前使用的配置

- `server/.env` - 后端当前使用的配置（由启动脚本自动复制）
- `client/.env` - 前端当前使用的配置（由启动脚本自动复制）

**注意**: `.env` 文件不会提交到 Git，每次启动时会自动从对应的环境配置文件复制。

## 使用方法

### 本地开发

```bash
./start.sh  # 自动使用 .env.local 配置
```

### 部署到 Dev 环境

```bash
# 在服务器上
cp server/.env.dev server/.env
cp client/.env.development client/.env
./deploy-dev.sh
```

### 部署到 Prod 环境

```bash
# 在服务器上
cp server/.env.production server/.env
cp client/.env.production client/.env
./deploy-prod.sh
```
